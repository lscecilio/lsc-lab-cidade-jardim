## CDJ — Sistema de Marketing | Agente Analista

## O QUE É ESTE PACOTE

Esta é uma **base de conhecimento estruturada** criada para turbinar um agente de IA
(Claude) com o conhecimento dos principais livros e pesquisas sobre marketing de luxo,
branding premium, neuromarketing, hospitalidade de alto padrão e mercado imobiliário
de luxo nacional e internacional.

Em vez de ter os PDFs dos livros, fizemos algo melhor: **pesquisamos e compilamos
os conceitos, frameworks, teorias e aplicações práticas de cada obra**, organizados
em arquivos de referência que o agente lê quando precisa.

## O ARQUIVO .SKILL

O arquivo `ESPECIALISTA-MARKETING-LUXO.skill` é a **skill do agente**.
Para usá-lo com o Claude (Cowork / Claude Code):

1. Abra o Cowork ou o Claude
2. Vá em **Plugins / Skills**
3. Clique em **Instalar skill**
4. Selecione o arquivo `ESPECIALISTA-MARKETING-LUXO.skill`

Depois de instalada, o Claude vai acionar automaticamente este especialista sempre
que você mencionar temas como: marketing de luxo, auditoria de marketing, branding
de alto padrão, loteamento fechado, HNWI, experiência de compra, tom de voz premium,
precificação de luxo, plantão de vendas, etc.

## COMO O AGENTE USA A BASE DE CONHECIMENTO

Quando acionado, o especialista:

1. **Usa os 10 frameworks internalizados** na skill (Kapferer, Kahneman, Cialdini,
   Ritz-Carlton, Neumeier, Pine & Gilmore, etc.) para responder perguntas imediatas

2. **Lê os arquivos das pastas relevantes** para aprofundar quando necessário
   — ex: se você pedir uma auditoria de posicionamento, ele vai ler
   `01-FUNDAMENTOS-DO-LUXO/` e `04-BRANDING-ESTRATEGICO/`

3. **Aplica o Protocolo de Auditoria em 8 Dimensões** para gerar diagnósticos
   estruturados com notas, red flags e plano de ação
