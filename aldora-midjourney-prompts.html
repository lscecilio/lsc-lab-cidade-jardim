Dicas para Melhores Resultados

  • Use --style raw para renders mais realistas e menos "painterly". Sem esse flag o Midjourney adiciona efeitos artísticos.
  • Para fachada verde escura exata: adicione "hex color #0a2416 dark forest green glass" ao prompt.
  • Para os detalhes dourados: "champagne gold #C8A84B metallic trim accents" garante consistência com a identidade AL'DORA.
  • Gere 4 variações de cada prompt e use Vary (Strong) para explorar diferentes composições.
  • Para imagens de marketing use --ar 9:16 (Story/Reels) e --ar 4:3 (impressos).
  • Use --no cars, --no people nas versões de fachada para renders mais limpos e atemporais.
  • Para referências de estilo, adicione: "inspired by One Hyde Park London, Fasano buildings, Cyrela premium".
  • Para os prompts de skyline, use --ar 21:9 para capturar toda a extensão do perfil escalonado.
  • Mantenha consistência: sempre mencione "14 towers" e "stepped ascending profile" nos prompts do masterplan.
