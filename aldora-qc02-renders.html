02 — Parâmetros

Configurações Recomendadas por Ferramenta

Midjourney v7

Aspect Ratio
--ar 16:9
Para vistas externas e áreas. Use --ar 9:16 para lobby/fachadas.
Style
--style raw
Mais fotorrealista. Evita o estilo "artístico" padrão do MJ.
Quality
--q 2
Máxima qualidade. Usa mais créditos mas entrega o melhor resultado.
Stylize
--s 200–400
Menor = mais fiel ao prompt. Para arquitetura, use 200–300.
Version
--v 7
Versão mais recente. Melhor em realismo arquitetônico.
Negative
--no cartoon
Adicione: --no cartoon, illustration, render, 3D

DALL-E 3 (via ChatGPT ou API)

Size
1792×1024
Landscape para vistas externas. 1024×1792 para fachadas.
Quality
hd
Sempre use quality=hd via API para máximo detalhe.
Style
natural
Evite "vivid" — produz saturação excessiva. Natural = fotorrealista.

Gemini Imagen 3 (via Google AI Studio)

Model
imagen-3.0
Melhor qualidade. Disponível via Google AI Studio e Vertex AI.
Aspect Ratio
16:9 / 4:3
Landscape para vistas externas. 3:4 para fachadas e interiores.
Negative Prompt
cartoon, CGI
Adicione: cartoon, CGI render, illustration, unrealistic lighting

💡 Dicas de Refinamento

  • Primeiro plano: Adicione "foreground blur" ou "shallow depth of field" para profundidade cinematográfica.
  • Pessoas: Use "people silhouettes" ou "blurred people" para escala sem perder foco no edifício.
  • Clima: "golden hour" = +30% de qualidade percebida. Evite luz do meio-dia (dura demais).
  • Upscale: No MJ, sempre use "Upscale (Subtle)" na imagem escolhida antes de exportar.
  • Referência: Cole uma foto do terreno real como referência visual no MJ (Image Prompt) para mais coerência.
  • Variações: Gere 4 variações de cada cena e escolha a melhor antes de upscale. Economiza créditos.

05 — Workflow

Fluxo de Produção de Renders AI

01
Prompt Base Copie o prompt da cena desejada acima. Ajuste para a ferramenta escolhida (MJ, DALL-E ou Gemini).
02
Gerar 4 Variações Sempre gere ao menos 4 variações. No MJ, use o botão V1-V4 na variação mais promissora.
03
Selecionar + Upscale Escolha a melhor variação e faça upscale (Subtle no MJ). Resolução mínima alvo: 3840×2160.
04
Pós-produção Use Adobe Photoshop ou Lightroom para: ajuste de cores, logo AL'DORA, remoção de erros AI.
05
Vídeo 3D Use Runway ML (Gen-3), Kling ou Pika Labs para animar o render estático. Duração ideal: 5–10s por cena.
06
Compilar Tour Monte um tour virtual com os 8 renders em sequência. Use DaVinci Resolve ou CapCut para edição final com trilha.

🎬 Ferramentas para Vídeo 3D / Animação

Runway ML Gen-3

Melhor para animar renders estáticos. Text-to-video e image-to-video. Recomendado.

Kling AI

Alternativa ao Runway. Bom para câmera orbital em edifícios. Plano gratuito disponível.

Pika Labs

Image-to-video com controle de câmera. Ideal para drone flyby simulado.

Luma Dream Machine

3D gaussian splatting. Transforma fotos em tours 3D interativos.
